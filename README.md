#khaleesicode

This is a flask application that handles my website. It features:
* Microblog application
* Login form
* Fried Chicken Index form
* Fried Chicken Index map of London using Bokeh charts

##Requirements
* Flask 0.10.1
* Python 2.7.9
* Flask-WTF
* Flask-Paginate
* SQLAlchemy
* Flask-SQLAlchemy
* Contextlib
* Bokeh 0.9.3
